##Chinese imperial exam grades (科舉功名) — the keju degree ladder [#34].##
# The keju (see se_QING_EXAM.txt) is the SOURCE of the empire's civil characters:
# every graduate carries exactly ONE of these mutually-exclusive status degrees.
# The merit ladder, ascending: 監生 jiansheng (purchased/academy floor) < 生員/秀才
# shengyuan < 舉人 juren < 貢士 gongshi < 進士 jinshi < 翰林 hanlin (the Academy apex).
# 繙譯進士 fanyi_jinshi is the parallel BANNER track — the Manchu/Mongol translation
# examination (翻譯科) that feeds the amban / resident-commissioner posts.
#
# The soft-preference in combined_stats_council_svalue (QING_governance_svalues.txt)
# scores a degree-holder far above a commoner wherever the council picks, so civil
# offices, ambanates, governorships and legations draw keju graduates by construction
# (the approved population-source model — governors/researchers/engine-diplomats are
# not script-appointable, so the keju gates the CANDIDATE POOL, not the seat).
#
# All are `type = status` and pairwise `opposites` (a man holds ONE functional degree).
# NOTE [#34 bugfix]: the old opposites lists named `jiansheng` and `gongsheng`, neither
# of which was ever declared — `jiansheng` is now defined below, and the stray
# `gongsheng` (a typo for the declared `gongshi`) is dropped from every list. Each
# degree now lists all six others, so the ladder is genuinely mutually exclusive.

# 監生 — the purchased / Imperial-Academy floor (捐納): a bought or academy rank, the
# lowest and least-respected qualification. Defines the trait the old opposites named
# but never declared, and models the sale-of-degrees track the exam pass-rate penalises.
jiansheng = {
	type = status
	holding_income_modifier = 0.01
	province = {
		local_upper_strata_happyness = -0.05
	}
	opposites = {
		shengyuan

}
