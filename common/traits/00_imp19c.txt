##Chinese imperial exam grades (科舉功名) — the keju degree ladder [#34].##
# The keju (see se_QING_EXAM.txt) is the SOURCE of the empire's civil characters:
# every graduate carries exactly ONE of these mutually-exclusive status degrees.
# The merit ladder, ascending: 監生 jiansheng (purchased/academy floor) < 生員/秀才
# shengyuan < 舉人 juren < 貢士 gongshi < 進士 jinshi < 翰林 hanlin (the Academy apex).
# 繙譯進士 fanyi_jinshi is the parallel BANNER track — the Manchu/Mongol translation
# examination (翻譯科) that feeds the amban / resident-commissioner posts.
#
# The soft-preference in combined_stats_council_svalue (QING_governance_svalues.txt)
# scores a degree-holder far above a commoner wherever the council picks, so civil
# offices, ambanates, governorships and legations draw keju graduates by construction
# (the approved population-source model — governors/researchers/engine-diplomats are
# not script-appointable, so the keju gates the CANDIDATE POOL, not the seat).
#
# All are `type = status` and pairwise `opposites` (a man holds ONE functional degree).
# NOTE [#34 bugfix]: the old opposites lists named `jiansheng` and `gongsheng`, neither
# of which was ever declared — `jiansheng` is now defined below, and the stray
# `gongsheng` (a typo for the declared `gongshi`) is dropped from every list. Each
# degree now lists all six others, so the ladder is genuinely mutually exclusive.

##Chinese military examination grades (武舉功名) — the martial degree ladder [#48].##
# The 武舉 (wǔjǔ, military examination) was the martial parallel to the civil keju: a
# THREE-tier ladder feeding the empire's Han officer corps of the Green Standard Army
# (綠營). Ascending: 武生員/武秀才 wu_shengyuan (licentiate, 院試) < 武舉人 wu_juren
# (provincial graduate, 鄉試) < 武進士 wu_jinshi (metropolitan/palace graduate, 會試+殿試),
# with 武狀元 wu_zhuangyuan (the single first-place palace laureate) as the apex.
#
# The exam was OUTER-FIELD first (mounted + foot archery, drawing the 8/10/12-力 bow,
# the 80/100/120-斤 great sword 舞刀, the 200/250/300-斤 lifting stone 掇石) then a light
# inner-field transcription of the 武經七書 — the Qianlong-24 (1759) reform tightened the
# martial tests and all but hollowed the written one, so by the 1763 start the ladder was
# a pure test of 弓馬技勇. Top palace graduates were made imperial bodyguards (侍衛): the
# 武狀元 a 一等侍衛, then out to a Green-Standard command. It was consistently LOWER in
# social prestige than the civil exam, and it was a HAN path — Manchu/Mongol officers rose
# through banner service and the translation examination (the fanyi_jinshi trait above).
#
# All are `type = status` and MUTUALLY EXCLUSIVE with the civil degrees and with each other
# (a man sat ONE track, held ONE functional degree): every military trait lists all six
# civil degrees AND the other three military ones as `opposites`, and the civil block does
# the reverse via [#48] additions. The martial soft-preference lives in
# qing_wu_degree_prestige_svalue (QING_governance_svalues.txt), which feeds ONLY the
# martial bench (council_sort_martial) so a 武進士 heads the war-minister / guard-commandant
# list the way a civil 進士 heads the civil offices.

# 武生員/武秀才 — the military licentiate (院試): passed the prefectural martial test and
# entered the local military school; the entry rung, modest prestige.
