### [#62 2026-07-23] NATIONALISM GROUP TRIGGERS (Design 4 — Culture → Nationalism)
#
# Nationalism is modelled at the culture-GROUP level (identity / loyalty / irredentism), while
# individual cultures remain the engine-level carriers of rights, names, portraits and plurality.
# These named triggers ARE the "nationalism" concept layer: each wraps the culture group(s) that
# constitute one nation, so game logic and the read-out can ask "is this a Zhonghua nation / a
# German nation" without renaming any culture key (the relabel is loc-only; internal keys untouched —
# renaming chinese_group->zhonghua would break name-gen/portrait coupling across ~34-59 files).
#
# Built on the PROVEN group primitives: country_culture_group (country scope),
# pop_culture_group (pop scope), dominant_province_culture_group (province scope — the .culture_group
# form does NOT parse, cf. se_QING_DECLINE.txt:684). Mirrors 00_culture_supergroups.txt exactly.
#
# 中華 Zhonghua minzu was coined by Liang Qichao in 1902 (Leibold 2007) — a period-authentic INVENTED
# pan-Han identity that awakens mid-game, unifying the Han sub-cultures under chinese_group. The other
# nations map one-to-one onto their existing culture groups.
#
# Naming: <nation>_nation_country_trigger / _pop_trigger / _province_trigger. No new state, pure reads.

# ---- 中華 ZHONGHUA (Han pan-nationalism) — the chinese_group cultures ------------------
zhonghua_nation_country_trigger = {
	country_culture_group = chinese_group
}
zhonghua_nation_pop_trigger = {
	pop_culture_group = chinese_group
}
zhonghua_nation_province_trigger = {
	dominant_province_culture_group = chinese_group
}

# ---- ANY-NATION helper: does this country hold a coherent single-group national identity? ----
# True when the country's primary culture group is one of the recognised nations. Used by the
# nationalism read-out to decide whether a country reads as a "nation" at all (vs a dynastic realm).
has_recognised_nationalism_country_trigger = {
	OR = {
		zhonghua_nation_country_trigger = yes
		manchu_nation_country_trigger = yes
		mongol_nation_country_trigger = yes
		german_nation_country_trigger = yes

		french_nation_country_trigger = yes
	}
}
