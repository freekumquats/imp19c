# [#312] SUCCESSION LAWS — female-line inheritance (Agnatic / Agnatic-Cognatic / Absolute-Cognatic).
#
# The mod's law rework removed the vanilla succession_law group entirely; this restores it as a
# selectable law group for monarchies, since the ONLY engine-supported way to change succession
# behaviour at runtime is `change_law = <succession_law>` (oracle-verified against Invictus
# events/monarchy_events.txt:2152 + epigoni.txt:235/273; there is NO set_succession / designate_heir
# effect). The engine maps the `succession = <type>` field to hardcoded C++ scoring that applies the
# per-type female penalty from defines NSuccession (FALLBACK_FEMALE_PENALTY for agnatic,
# COGNATIC_FEMALE_PENALTY for cognatic).
#
# THREE TIERS (task #312):
#   • Agnatic (男系繼承) — succession = agnatic. Males only in practice (heavy female penalty).
#   • Agnatic-Cognatic (男系優先) — succession = cognatic. Males preferred; a daughter inherits
#       only if there is no eligible male line (the historical norm for most European monarchies).
#   • Absolute-Cognatic (絕對繼承) — the engine has NO fourth succession type and only three global
#       female-penalty constants, so TRUE gender-equal succession is not per-country scriptable
#       (oracle finding E). It is modelled as the cognatic law PLUS the absolute_cognatic_succession
#       country modifier (granted by the reform decision): mechanically still cognatic, but the modifier
#       + tooltip present it as equal succession and grant a small legitimacy/stability nod for the
#       modernised, uncontested line. See DEFERRED note in overnight_decisions3.md.
#
# Restricted to monarchies (potential is_monarchy = yes), like the vanilla group. Republics/tribes
# never see it. No allow-block restrictions on the base tiers (the reform DECISION carries the cost +
# the Pragmatic-Sanction contest); egyptian/seniority/elective vanilla types are intentionally NOT
# reintroduced (out of scope — this task is female-line laws only).
succession_law = {
	potential = {
		is_monarchy = yes
	}

	# Male-line only (the de-facto Salic default). First tier so it is the group default.
	agnatic_succession_law = {
		allow = {
		}
		succession = agnatic
	}

	# Males preferred, a daughter inherits if the male line fails (Habsburg/most-European norm).
	cognatic_succession_law = {
		allow = {
		}
		succession = cognatic
	}
}
