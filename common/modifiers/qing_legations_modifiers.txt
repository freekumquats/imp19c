########################################################################
# qing_legations_modifiers.txt — modifiers for the diplomatic missions and
# permanent legations (遣使 / 駐外使館). See se_QING_LEGATIONS.txt.
#
# One lasting country modifier per permanent legation (a resident minister in
# that power's capital) plus the Burlingame Mission's temporary modifier. Each
# legation is a standing gain in diplomatic reach; together they raise the
# dynasty's ability to conduct modern diplomacy. The tension easing + prestige +
# reform-pressure cost are applied as nudges in the effects, not here.
#
# All keys verified present in common/modifiers/00_*.txt. Country scope.
# Prefix: qing_legation_ / qing_burlingame_
#
# Sources: Spence, The Search for Modern China; Immanuel Hsü, China's Entrance
#   into the Family of Nations (1960).
########################################################################
