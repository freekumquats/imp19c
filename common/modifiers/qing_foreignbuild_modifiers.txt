# qing_foreignbuild_modifiers.txt — [#52 2026-07-23] the COUNTRY-level standing
# modifiers hung off the foreign-building family (Design 1). Per the concrete-over-
# abstract rule, these are NOT baked into the buildings (buildings carry only LOCAL
# province/state modifiers); they are applied to the country by the authorization
# effects in se_QING_FOREIGNBUILD.txt while the matching building stands, and removed
# when it is torn down. Prefix: qing_fbuild_. Country scope.
#
#   qing_fbuild_works_patron       the Qing sponsors a subject's 洋務局 industrialization —
#                                  standing prestige/diplomatic lift as the patron of reform.
#   qing_embassy_representation     a resident Qing legation abroad (使館) — a standing
#                                  diplomatic-reputation + influence lift (Westphalian presence).
#   qing_treaty_port_humiliation    a foreign-held treaty port stands on Qing soil — a standing
#                                  reputation + unrest drag (the unequal-treaty grievance).
#
# All keys verified present in common/modifiers/00_*.txt / qing_embassy_modifiers.txt
# (diplomatic_reputation, monthly_political_influence, global_unrest, research_points_modifier,
# global_monthly_civilization). Sources: Design 1 digest (Hevia 1995; Jackson 2020; Elman 2004).
########################################################################
