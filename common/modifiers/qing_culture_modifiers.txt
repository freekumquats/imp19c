# qing_culture_modifiers.txt — cultural-happening modifiers for the Qing
# player-experience suite (task #15; qing_culture_events.txt).

# The great cultural monuments of the High Qing, offered anachronistically
# to the 1815 player: the imperial dictionaries and encyclopaedias, the
# vernacular novel, the literary inquisition, and the luxury crafts (jade,
# porcelain, silk, painting) that made "China" a byword for refinement.
#
# All keys validated against the mod's country/culture modifier vocabulary
# (research_points_modifier, global_population_happiness, culture/omen keys,
# global_pop_assimilation_speed_modifier, monthly_legitimacy, commerce).
# Prefix = qing_culture_ ; durations set by the events that add them.
########################################################################
