########################################################################
# 00_napoleon.txt — LA GRANDE ARMÉE, the Napoleonic military tradition made
# available to the Qing ONLY when Napoleon holds the Emperor Emeritus office
# (task #65). Where the Manchu tree (00_manchu.txt) commemorates the Ten Great
# Campaigns, this tree imports the doctrine that made Napoleon the age's foremost
# soldier — and, per the user's brief, its buffs are DELIBERATELY OVERSIZED
# (roughly 1.5–2× the manchu tree's) to model a genuine revolution in the art of
# war dropped into an army that had none.
#
# GATING — this is the point the user was explicit about: the tree is NOT gated
# on a culture group (Napoleon is French, the Qing are jurchen — a culture gate
# could never open it). It is gated on the COUNTRY VARIABLE `qing_napoleon_present`,
# set by QING_napoleon_spawn in se_QING_NAPOLEON.txt. So the tree is reachable by
# exactly one country in exactly one timeline: the Qing that received Napoleon.
#
# Four theatres + a capstone, the pillars of the Napoleonic system:
#   A  LA GARDE IMPÉRIALE   — the Imperial Guard: elite heavy infantry, the
#                             decisive reserve committed to break the line.
#   B  LE DIEU DE LA GUERRE — massed artillery: "c'est avec des canons qu'on fait
#                             la guerre." Napoleon began as a gunner; the grand
#                             battery is his signature. (Drives the now-real
#                             artillery arm directly via artillery_* keys, alongside
#                             siege / assault power — see army_artillery.txt, #89.)
#   C  LA MANŒUVRE          — the corps system and the forced march: bataillon
#                             carré, march-divided-fight-united, la manœuvre sur
#                             les derrières.
#   D  LE MARÉCHALAT        — the marshalate: Napoleon's marshals, loyal and
#                             brilliant, and the levée that filled the ranks.
#   Capstone  L'EMPEREUR    — requires the final node of all four theatres.
#
# Icons are PLACEHOLDERS pointing at existing .dds in the shared tradition icon
# set, exactly as 00_manchu.txt does, so no node renders blank. Only modifier keys
# proven valid elsewhere in this mod are used. This is a FLAT tree (arabic / indian
# / japanese / manchu pattern): no path-container nodes, so each theatre's capstone
# reward is folded into that theatre's final node's `modifier` block rather than a
# separate `bonus = {}` (which a flat tree silently drops).
#
# Sources: standard Napoleonic historiography (the corps d'armée, the grand
# battery, the ordre mixte, the marshalate); see [[imp19c-napoleon-in-china]].
########################################################################

napoleon_grande_armee = {
	color = hsv { 0.62 0.75 0.85 }
	image = arabic_philosophy_start_bonus   # placeholder start image (existing .dds)

	allow = {

		# is a country trigger; the tradition `allow` block is evaluated in country
		# scope, so this reads the flag QING_napoleon_spawn set on CHI.
		trigger_if = {
			limit = {
				has_variable = qing_napoleon_present
			}
			custom_tooltip = {
				text = "napoleon_tradition_requires_emeritus"
			}
			has_variable = qing_napoleon_present
		}
	}

	## START ####################################################################
	# LA GRANDE ARMÉE — the army reorganised into self-contained corps, each a
	# combined-arms army in miniature. The opening bonus is the morale and reserve
	# of a nation in arms.
	napoleon_start = {
		icon = indian_philosophy_start_bonus   # placeholder (existing .dds)
		ai_will_do = {
			modifier = {

			land_morale_modifier = 0.10
			global_manpower_modifier = 0.10
		}
	}

	# The gunner-emperor's own arm — and, with task #89, the centrepiece of the whole

	# the enemy centre with it before the assault went in. The artillery unit is now a
	# real, recruitable arm (see army_artillery.txt), so this path drives the field
	# guns DIRECTLY through the artillery_* keys, alongside the battering power
	# (assault / siege) that models the grand battery's effect on the whole army.

	# L'École de Brienne — the artillery officer's schooling: the mathematics of
