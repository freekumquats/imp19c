# [#61 2026-07-23] IDEOLOGIES AS RELIGIONS (Design 3 — Religion → Ideology).
#
# Modern political ideologies modelled as "political religions" (Gentile, Politics as Religion 2006;
# Voegelin) — comprehensive worldviews with sacred texts, prophets (thinkers), conversion and schism.
# They are added as NEW religion entries that COEXIST alongside the historical faiths (the coexistence
# decision, Design 3 §F2): a country/pop is either on a faith OR has "converted" to an ideology, both
# occupying the same engine `religion` slot but distinct entries. This collapses the blast radius — NO
# mass setup/main/00_default.txt remap is needed; the overwhelming majority of religion= lines stay on
# their historical faith untouched, and only a small set of 1763-nascent holders are reseeded later.
#
# religion_category = pantheon (SIBLING to confucianism / the faiths). These carry NO custom deities in
# this chunk: the generic deity set (common/deities/00_generic.txt) covers any religion lacking its own,
# so an ideology religion is complete + panel-safe with zero custom-deity DB registration (the DB-key-
# contiguity crash class is thereby avoided entirely). A custom thinker-pantheon (2 deities × 4 categories
# per ideology, from the Design 3 §A.3 roster) is a SEPARATE reviewable follow-on, not this chunk.
#
# can_deify_ruler = no (an ideology's figureheads are theorists, not deified rulers — matches confucianism).
# Adoption is the proven layered set_country_religion flow (se_QING_IDEOLOGY.txt). Coexistence with the
# vanilla state_secularism/state_atheism laws is reconciled by excluding ideologies from those laws' allow.
#
# Sources: Design 3 digest (Schwartz 1964 on Yan Fu; Levenson 1953 on Liang Qichao; Anderson; Berlin).
