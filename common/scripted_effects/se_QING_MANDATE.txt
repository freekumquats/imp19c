# se_QING_MANDATE.txt — [#23 §2b Cluster A 2026-07-20] THE MANDATE OF HEAVEN (天命) disaster layer.
#
# Design doc: DESIGN_qing_vanilla_layering.md §2b Cluster A. In Qing political theology a natural
# calamity reads as Heaven's displeasure (天譴): a disaster is not merely an economic hit, it is a
# judgement on the dynasty's fitness to rule. So this layer BUILDS ON the vanilla legitimacy/stability
# values the mod already writes everywhere (§1a) — a disaster nudges those values down, and the
# ruler's RESPONSE (relief vs. neglect) trades treasury against legitimacy. One shared effect unifies
# every disaster behind a single Qing-thematic hook instead of five disconnected vanilla chains.
#
# This is the PILOT: the shared qing_mandate_omen effect + one disaster (fire/conflagration 火災).
# food/storms/comet/natural_disasters are DEFERRED to post-boot batches (design doc) — each is a
# separate events file wired onto this same effect.
#
# Scope: COUNTRY (ROOT = CHI) for qing_mandate_omen. se_LOG sys = QING. Prefix = QING_mandate_.
# Proven idioms only: add_stability / add_legitimacy (country, cf. se_QING_PRINCES 306-307),
# add_province_modifier { duration } (cf. se_QING_BURMA 47-51), add_treasury (cf. se_QING_CANTON 140).
# No create_character, no add_trait, no sorting iterator — nothing in the boot-crash catalogue.
########################################################################

# ======================================================================
# qing_mandate_omen — the shared disaster→Mandate hook. Country scope (ROOT = CHI).
# Params:
#   $severity$ = 1 (minor) | 2 (major) | 3 (grave). Scales the legitimacy/stability delta.
# Applies the vanilla legitimacy/stability judgement (the "Heaven's displeasure" read). The
# LOCAL economic effect + the ruler-facing relief-vs-neglect choice live in the calling disaster
# event (which knows the affected province); this effect is only the shared Mandate consequence
# so every disaster scores legitimacy the same way. Self-contained, idempotent per call.
# ======================================================================
qing_mandate_omen = {
	# minor 天譴: a warning. legitimacy dips, stability barely.
	if = {
		limit = { $severity$ = 1 }
		add_legitimacy = -3
	}
	# major 天譴: a clear judgement. legitimacy + stability both bite.
	else_if = {
		limit = { $severity$ = 2 }
		add_legitimacy = -6
		add_stability = -2
	}
	# grave 天譴: Heaven's wrath. the dynasty's fitness is openly doubted.
	else_if = {
		limit = { $severity$ = 3 }
		add_legitimacy = -10
		add_stability = -4
	}
	LOG_line = { sys = QING  msg = "mandate: 天譴 — a calamity is read as Heaven's judgement (severity above) for" }
}
